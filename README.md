# Crazy Bike Shop with Durable Task Scheduler

The Durable Task Scheduler brings managed, reliable stateful orchestration to Azure, now extending beyond functions to orchestrate Container Apps and other compute resources. With its .NET SDK, you can define and coordinate complex workflows across distributed microservices, leveraging built-in monitoring and fault tolerance. In this session, we’ll demonstrate how to implement robust orchestration patterns for Container Apps using the Durable Task framework in .NET, enabling scalable cloud-native architectures with minimal overhead.
